# Genie Diagnostics Website Files

This package contains all the source code files for the Genie Diagnostics lab management software website.

## Files Included:

### 1. index.html
The main HTML file containing the complete website structure including:
- Navigation header with Genie Diagnostics logo
- Hero section with key statistics and call-to-action
- Features section showcasing lab software capabilities
- Pricing plans (Basic ₹599, Standard ₹749, Advanced ₹999)
- Customer testimonials
- FAQ section with accordion functionality
- Contact information

### 2. style.css
Comprehensive CSS stylesheet with:
- Modern design system with CSS custom properties
- Professional medical/diagnostic theme (blue, teal, white colors)
- Responsive design for all device sizes
- Smooth animations and transitions
- Accessibility-friendly styles
- Professional typography and spacing

### 3. app.js
JavaScript functionality including:
- Smooth scrolling navigation
- Mobile menu toggle
- FAQ accordion interactions
- Animated statistics counters
- Form validation
- Notification system
- Keyboard navigation support
- Performance monitoring

## Setup Instructions:

1. Download all three files to the same folder
2. Ensure the files are named exactly as listed above
3. Open index.html in a web browser
4. For best results, serve through a local web server

## Dependencies:

The website uses Font Awesome icons loaded from CDN:
```html
<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css" rel="stylesheet">
```
